Getting Started

1

Get an OpenAI API Key

Sign up at platform.openai.com and create an API key. Add some credit to your account.

2

Create a Connection Profile

Open TalkAlly, go to Settings → Connection Profiles → Add Profile. Choose Personal mode and paste your API key.

3

Set Up Your Personal Profile

Add your name and configure Hands-Free Contacts for quick voice calling.

4

Start Talking!

Tap VOICE to connect and start speaking. Try "What's the weather?" or "Call [contact name]".
